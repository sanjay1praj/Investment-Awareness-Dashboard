# Investment-Awareness-Dashboard
Interactive Power BI dashboard analyzing investment awareness and investment patterns among young investors.

## Business Objective
Analyze the awareness, preferences, and investment patterns of young investors using an interactive Power BI dashboard.

## Tools Used
- Power BI
- Excel
- DAX

## Key Features
- Investment Preference Analysis
- Demographic Analysis
- Risk Tolerance Analysis
- Investment Awareness Insights
- Interactive Filters and Slicers
